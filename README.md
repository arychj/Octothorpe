# Octothorpe
Octothorpe is rules engine and services framework. Basically it is a partial IFTTT clone which you can use to create custom services that can optionally spawn events that other services can consume via rule mappings while also providing real-time synchronous responses to the injector.

---

#### Requirements
* Python 3.6

#### Usage
Setup:
```bash
./run.py --config config.xml --setup
```

Running the server:
```bash
./run.py --config config.xml
```
